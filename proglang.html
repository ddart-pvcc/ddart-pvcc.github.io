Programming Languages

TERM DEFINITION
Machine Language Programs for modern computers consist of sequences of instructions that are encoded as numeric digits.
Debugging Locating errors in code that must be corrected.
Assembler Convert mnemonic expressions into machine language instructions.
Assembly Language Mnemonic system for representing programs.
Translator Translates programs expressed in these high-level primitives into machine-language programs.
Compiler Compile several machine instructions into short sequences to simulate the activity requested
by a single high-level primitive.
Interpreter Executes the instructions as they were translated instead of recording the translated version for future use.
Cross-Platform Software Software that is independent of an operating system’s design as well as the machine’s
hardware design and is therefore executable throughout a network.
Natural Language Spoken languages of the world such as English, German, Spanish, and Mandarin.
Formal Language Programming languages such as Python, C++, Java, and COBOL.
Machine Independence Languages that do not refer to the attributes of any particular machine.
They can be compiled as easily for one machine as for another.
High-Level Programming Language Provide more abstractions for the programmer and make it easier for people to read and write a program. They leverage millions
of lines of library code, invoke built-in operating system functions, or initiate complex interactions with networks and databases.
Object-Oriented Programming (OOP) Software system is viewed as a collection of units, each of which is capable of performing the actions
that are immediately related to itself as well as requesting actions of other objects.
Scripting Language A subset of the imperative languages. They are typically used to perform
administrative tasks rather than to develop complex programs.
Data Types:
  • Integer
  • Float
  • String
     
  • Numeric data consisting of whole numbers.
  • Numeric data that might contain values other than whole numbers stored in floating-point notation.
  • Text data.
Primitive Data Types Data types that are included as primitives in a programming language, such as int for integer and char for character.
Data Structure Conceptual shape or arrangement of data.
Python Collection Data Structures
  • List
  • Tuple
  • Set
  • Dictionary
     
  • Used to store multiple items in a single variable. Created using square brackets.
  • Used to store multiple items in a single variable. Ordered and unchangeable. Created using round brackets.
  • Used to store multiple items in a single variable. Unordered, unchangeable, and unindexed. Created using curly brackets.
  • Used to store data in key:value pairs. Ordered, changeable, does not allow duplicates. Created using curly brackets.
Function Set of instructions for performing a task that can be used as an abstract tool by other program units.
Local Variable A variable declared within a function. It can be referenced only within that function.
Global Variable Scope not restricted to a particular part of a program.
Source Program Program in its original form before a translation.
Object Program Translated version of a program.
